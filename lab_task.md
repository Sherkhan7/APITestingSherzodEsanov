# Laboratory Task: Web API Testing with RestAssured

## Objective

Implement **any 10 tests** using the Swagger Petstore API:
<https://petstore.swagger.io/>

## Requirements

- Tests should include **both positive and negative** scenarios.
- Tests should include validations covering:
  - Response status code
  - Content-Type header
  - Data values
  - Data structure
